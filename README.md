## Key Features

* Front Page Graphs and Widgets
  - Graphs can display the active users throughout the months of a year
  - Widgets display additional information about the project's sales percentage compared to last month, costs, recent users, and recent orders
  - <img width="1085" alt="admin1" src="https://github.com/user-attachments/assets/b0db2893-ad89-4eba-a179-d65ca8ae27f1">

* User Page
  - A chart displays all the users from the API its stored
  - Can add new users, update existing ones, or delete users
* Product Page
  - A chart is displayed to show all current products the project has
  
  - <img width="1099" alt="admin2" src="https://github.com/user-attachments/assets/20556d60-e195-4a17-8ed0-cf0f96e9b40c">
  - Can add new products, update exisiting ones, and delte users
  - <img width="1090" alt="admin3" src="https://github.com/user-attachments/assets/461dd74e-2269-4bbe-bd05-590d187e8d9f">


*Both User and Product Changes Are Sent to API
* Viewing Transactions
  - A chart is displayed to show all orders that have been made
  - Can delete orders
* Additional Options In Sidebar In Progress

## How To Use

Please make sure to run the backend project first and following the instructions listed on the page. Afterward download the current files and create an enviroment file with the following variables:<br></br>
API_URL = "https://localhost:5000/api/"<br></br>
Afterward just run 
- npm build
- npm run
